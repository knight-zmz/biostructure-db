## 1️⃣ 域名配置

### 推荐域名
```
biostructure.cn      (推荐)
pdb-cloud.com        (推荐)
biopro.cn            (简短)
structuredb.io       (技术感)
```

### 配置步骤

**你需要做的:**
1. 购买域名 (阿里云/腾讯云)
2. 添加 DNS 解析:
   ```
   类型：A
   主机记录：@
   记录值：101.200.53.98
   TTL: 10 分钟
   ```
3. 添加 www 解析:
   ```
   类型：CNAME
   主机记录：www
   记录值：biostructure.cn
   TTL: 10 分钟
   ```

**我会自动做的:**
1. 配置 Nginx 域名
2. 申请 Let's Encrypt 证书
3. 配置 HTTPS 自动续期
4. HTTP 自动跳转 HTTPS
