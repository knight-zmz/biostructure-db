# 2026-03-27 学习记录

## 🧬 生物数据库最佳实践

### 参考项目

#### 1. Molstar (PDBe + RCSB PDB)
- **GitHub**: https://github.com/molstar/molstar
- **用途**: 专业 3D 分子可视化
- **特点**:
  - 支持 PDB, mmCIF, SDF, MOL2 等格式
  - 支持体积数据（EMDB）
  - 高性能 WebGL 渲染
  - 支持序列注释
  - 支持结构比对

#### 2. RCSB PDB API
- **文档**: https://data.rcsb.org/rest/v1/
- **端点**:
  - `/rest/v1/core/entry/{pdbId}` - 结构信息
  - `/rest/v1/core/uniprot/{pdbId}` - UniProt 映射
  - `/rest/v1/core/chain/{pdbId}/{chainId}` - 链信息

#### 3. UniProt API
- **文档**: https://www.uniprot.org/help/api
- **端点**:
  - `/uniprotkb/{accession}.json` - 蛋白质信息
  - `/uniref/{id}` - 序列聚类

### 数据库架构学习

#### PDB 数据结构
1. **结构数据**
   - PDB ID (4 字符)
   - 标题、摘要
   - 实验方法 (X-Ray, NMR, EM, ...)
   - 分辨率、R-factor

2. **序列数据**
   - 链 ID (A, B, C, ...)
   - 氨基酸序列
   - 二级结构
   - Pfam 结构域

3. **原子数据**
   - 原子名称 (CA, CB, N, O, ...)
   - 坐标 (x, y, z)
   - B-factor (温度因子)
   - 占有率

4. **注释数据**
   - UniProt 映射
   - 配体结合位点
   - 活性位点
   - 金属离子
   - 翻译后修饰

## 🔍 数据库查询优化

### 索引策略
- 主键索引: 所有表
- 外键索引: pdb_id, chain_id
- 全文索引: 序列 (使用 pg_trgm)
- 复合索引: (pdb_id, chain_id)

### 查询优化
1. 使用 `LIMIT` 限制结果集
2. 使用 `LEFT JOIN` 减少 N+1 查询
3. 使用 `COALESCE` 处理 NULL 值
4. 使用 `ARRAY_AGG` 聚合数据
